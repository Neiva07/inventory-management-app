# Stockify Runtime Logs MCP Wrapper

Thin client wrapper over the internal runtime log API.

Required environment for callers:

- `STOCKIFY_RUNTIME_LOGS_BASE_URL`
- `RUNTIME_LOGS_INTERNAL_API_KEY`

Tools represented in v1:

- `search_runtime_logs`
- `get_runtime_log_launch`
- `export_runtime_log_launch`
